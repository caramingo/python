Клиент серверное приложение на python для выполнения bash команд без использования ssh.

На сервер должне быть утсновлен python3, запускать серверную часть вот так python3 server.py
Учти что сервер работает на 5000 порту, он может быть закрыт фаерволлом.

Для работы клиента нужно установить apt install python3-pyqt5 или pip install PyQt5
Укажи IP сервера в client.py в строке SERVER_HOST = 'your.server.ip'
Так же на клиентской части сменить токен AUTH_TOKEN = 'mysecrettoken' на что  серьезное.
Для запуска на клиенте выполни python3 client.py 'uptime'
Вместо uptime может быть любая другая не привилигированная команда.
